# React Kanban Demo

Reactで作成されたカンバンボードです。

## アプリのURL

https://react-kanban-demo.vercel.app/

## 機能

* 未着手 / 対応中 / 完了カテゴリのカンバンボード
* ドラッグ＆ドロップによるタスクの移動
* タスクの新規作成
* タスクの編集
* タスクの削除

## 開発環境

* JavaScript
* Yarn
* React
* dnd kit
* ESLint
* Prettier
* Tailwind CSS
* Visual Studio Code

## 利用方法

```sh
git clone https://github.com/ot07/react-kanban-demo.git
cd react-kanban-demo
yarn install
yarn start
```
